# Bookmarks MERN Project

![](https://i.imgur.com/9IZ6ZPz.png)

## Steps

**Start the dev server**
```bash
npm run dev
```

**Build files for production**
```bash
npm run build
```

**Deploy for production script**
```bash
npm start
```

### Dev Environment: 
Gulp, Webpack, Browsersync, Nodemon and Babel
